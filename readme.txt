A lightweight and powerful file manager for WordPress admins. Browse, create, upload, edit, and manage files and folders with a clean UI.

== Description ==

**QB File Manager by QuietBolt** is a simple yet powerful admin tool to manage files and folders directly from your WordPress dashboard.

**Key Features:**
- 📁 Folder browsing from root directory  
- 🗂️ Create folders  
- 📄 Create files  
- 📤 Upload files  
- ✏️ Edit file contents  
- 🗑️ Delete files and folders  
- 🧩 Manage common file types  
- ✅ Clean and modern UI  
- 🔒 Admin-only access  

**Supported File Types:**

- **Editable Files:** `php`, `html`, `css`, `js`, `txt`, `json`, `xml`, `md`, `log`, `ini`, `.htaccess`, `svg`, `csv`, `yml`  
- **Image Files:** `jpg`, `jpeg`, `png`, `gif`, `webp`, `ico`, `bmp`  
- **PDF Files:** `pdf`  
- **Archives:** `zip`, `tar`, `gz`, `rar`  

This plugin is designed for developers, admins, and power users who need quick file access without leaving the WordPress backend.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/qb-file-manager/`  
2. Activate the plugin through the ‘Plugins’ menu in WordPress  
3. Go to **QB File Manager** in your WordPress dashboard  

== Frequently Asked Questions ==

= Who can use this plugin? =  
Only users with administrator privileges can access the file manager.

= Can I edit PHP and other code files safely? =  
Yes, but it's recommended to always back up your site before editing critical files.

= Does this plugin allow access to the entire server? =  
No, it only provides access to files and folders within your WordPress installation directory.

= Does it have a file preview? =  
Not currently. Files are opened in an inline code editor.
